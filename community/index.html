    Support¶

    Siddhi an open-source project under Apache License built by hundreds of contributors. We always welcome new members to the community and appreciate their support.

    You can get help on Siddhi in the following ways.

    Community Collaboration¶

    • For architecture and code related discussions with developers, please initiate a discussion on the Siddhi-Dev Google Group.

    • For more detail discussions on Siddhi usage and get help on implementing use cases, initiate a discussion on the Siddhi-User Google Group.

    • Siddhi Slack chanel is used to collaborate with the project users, committers and contributors.
      If you are not already a member please feel free to request an invitation by giving your email in the below dialog box.

    • Users can also ask questions from the developers using the Siddhi GitHub issue tracker with the question label.

    • Users can ask questions and get support via StackOverflow, make sure to add the siddhi tag to the issue for better response.
